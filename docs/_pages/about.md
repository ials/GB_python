# Course information
## Description

The laboratory aims to provide the necessary basis for learning how to manage, analyze and visualize geospatial data through open source tools (geospatial libraries for python, qgis, R â€¦)

At the end of the course, students will be able to:

- understand the specificity of the geospatial data model
- elaborate and integrate geospatial data (vector and raster)
- create maps (also accessible via the web)
